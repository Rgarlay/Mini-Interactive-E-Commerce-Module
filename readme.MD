# Mini-Interactive-E-Commerce-Module

A lightweight, full-stack e-commerce application featuring a dynamic product gallery and a functional checkout system. This project demonstrates state management in vanilla JavaScript and basic CRUD operations using Node.js and MongoDB.

---

## 🚀 Features
* **Dynamic Product Listing:** Products are served via the backend and rendered dynamically on the storefront.
* **Two-Page Navigation:** Separate dedicated pages for browsing products and managing the checkout process.
* **Interactive Shopping Cart:** * Add multiple products to the cart.
    * Adjust quantities (increment/decrement) in real-time.
    * Automatic calculation of totals.
* **Persistent Backend:** Uses Express and MongoDB to manage product data and order logic.

---

## 🛠️ Tech Stack

### Frontend
* **HTML5 & CSS3:** Structured layout and responsive styling.
* **Vanilla JavaScript:** DOM manipulation and cart state management.

### Backend
* **Node.js:** JavaScript runtime environment.
* **Express.js:** Web framework for building the API routes.
* **MongoDB:** NoSQL database for storing product information.
* **Mongoose:** ODM (Object Data Modeling) library for MongoDB and Node.js.
